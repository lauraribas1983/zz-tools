REQUISITO ÚNICO: Python 3.10+ instalado en el PC.
  Descarga: https://www.python.org/downloads/

INSTALACIÓN (primera vez — solo una vez):
  Doble clic en: INSTALAR_DEPENDENCIAS.bat
  Espera ~2 minutos.

═══ ARRANCAR LAS HERRAMIENTAS ═══════════════════

Windows:  Doble clic en INICIAR_HERRAMIENTAS.bat
Mac/Linux: Abre terminal → bash INICIAR_HERRAMIENTAS.sh

═══ ACCESO DESDE MÓVIL ══════════════════════════

1. El PC y el móvil deben estar en la misma WiFi.
2. Arranca las herramientas en el PC.
3. En el PC: abre cmd → escribe "ipconfig"
   Busca "Dirección IPv4": suele ser 192.168.1.XXX
4. En el móvil abre el navegador:
   http://192.168.1.XXX:8501  → TechPack Generator
   http://192.168.1.XXX:8502  → Image Processor

Consejo: Guarda esas URLs como accesos directos
en el móvil (añadir a pantalla inicio).

═══ LAS HERRAMIENTAS ════════════════════════════

📋 TECHPACK GENERATOR (puerto 8501)
   Rellenas el formulario → genera PDF ficha técnica
   profesional listo para mandar a fábrica.
   No se necesita IA. No se necesita internet.
   PDF incluye: descripción, tejido, medidas por talla,
   gráfica, construcción, etiquetado y QC.

🖼️ IMAGE PROCESSOR (puerto 8502)
   Subes imágenes de renders (desde móvil también).
   Genera automáticamente:
   - Fondo blanco (ghost mannequin)
   - Versión web 1200px
   - Versión catálogo 2400px alta calidad
   - Versión techpack 800px
   - Miniatura 400px para WhatsApp/redes
   Todo se descarga en un ZIP con carpetas organizadas.
